1. <span data-ttu-id="6344c-116">Seleccione el icono ![Buscar página o informe](media/ui-search/search_small.png "icono Buscar página o informe"), escriba **Selección informes ventas** y, a continuación, seleccione el vínculo relacionado.</span><span class="sxs-lookup"><span data-stu-id="6344c-116">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Report Selections Sales**, and then choose the related link.</span></span>

3. <span data-ttu-id="6344c-118">En una línea nueva, en el campo **Id. informe**, seleccione, por ejemplo, informe estándar 1306.</span><span class="sxs-lookup"><span data-stu-id="6344c-118">On a new line, in the **Report ID** field, select, for example, standard report 1306.</span></span>

    <span data-ttu-id="6344c-121">Las plantillas de informes definen tanto el estilo como el contenido del cuerpo del correo, incluyendo el texto estándar que precede la información principal del documento.</span><span class="sxs-lookup"><span data-stu-id="6344c-121">Report layouts define both the style and the content of the email body, including the standard text that precedes the core document information in the email body.</span></span> <span data-ttu-id="6344c-122">Puede ver todos los diseños de informe disponibles si elige el botón **Seleccionar de la lista completa** en la lista desplegable.</span><span class="sxs-lookup"><span data-stu-id="6344c-122">You can see all available report layouts if you choose the **Select from full list** button in the drop-down list.</span></span>
6. <span data-ttu-id="6344c-123">Para ver o editar el diseño en el que se basa el cuerpo del correo electrónico, seleccione el diseño en la ventana **Diseños de informe personalizados** y, a continuación, elija la acción **Editar diseño**.</span><span class="sxs-lookup"><span data-stu-id="6344c-123">To view or edit the layout that the email body is based on, select the layout in the **Custom Report Layouts** window, and then choose the **Edit Layout** action.</span></span>
7. <span data-ttu-id="6344c-124">Si desea ofrecer a sus clientes el pago de las ventas de forma electrónica puede configurar el servicio de pago relacionado, como por ejemplo PayPal y, a continuación, puede disponer también de su información y su hipervínculo en el cuerpo del correo.</span><span class="sxs-lookup"><span data-stu-id="6344c-124">If you want to offer customers to pay for sales electronically, you can set up the related payment service, such as PayPal, and then have the PayPal information and hyperlink inserted in the email body as well.</span></span> <span data-ttu-id="6344c-125">Para obtener más información, consulte [Permitir pagos de cliente a través de PayPal](sales-how-enable-payment-service-extensions.md).</span><span class="sxs-lookup"><span data-stu-id="6344c-125">For more information, see [Enable Customer Payments Through PayPal](sales-how-enable-payment-service-extensions.md).</span></span>
8. <span data-ttu-id="6344c-126">Elija el botón **Aceptar**.</span><span class="sxs-lookup"><span data-stu-id="6344c-126">Choose the **OK** button.</span></span>

<span data-ttu-id="6344c-127">Ahora bien, cuando selecciona, por ejemplo, la acción **Enviar** en la ventana **Factura venta reg.**, el cuerpo del correo electrónico contendrá la información del documento del informe 1306 precedida por el texto estándar siguiendo el diseño de informe que seleccionó en el paso 5.</span><span class="sxs-lookup"><span data-stu-id="6344c-127">Now, when you choose, for example, the **Send** action in the **Posted Sales Invoice** window, the email body will contain the document information of report 1306 preceded by styled standard text according to the report layout that you selected in step 5.</span></span>
