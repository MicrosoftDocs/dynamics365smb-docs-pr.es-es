# <a name="import-and-export-a-report-or-document-layout"></a><span data-ttu-id="54355-103">Importar y exportar un diseño de informe o documento personalizado</span><span class="sxs-lookup"><span data-stu-id="54355-103">Import and Export a Report or Document Layout</span></span>
<span data-ttu-id="54355-104">Puede importar y exportar un diseño de informe personalizado existente como archivo a o desde una ubicación de su equipo y red.</span><span class="sxs-lookup"><span data-stu-id="54355-104">You can import and export an existing custom report layout as a file to and from a location on your computer and network.</span></span> <span data-ttu-id="54355-105">Por ejemplo, puede exportar un diseño de informe y, a continuación, enviar el archivo a otra persona para que lo modifique.</span><span class="sxs-lookup"><span data-stu-id="54355-105">For example, you can export a report layout, and then send the file to another person to modify.</span></span> <span data-ttu-id="54355-106">Esa persona podrá entonces hacer las modificaciones en el diseño y devolverle el archivo para que pueda importarlo de nuevo.</span><span class="sxs-lookup"><span data-stu-id="54355-106">That person can then make the modifications to layout and return the file to you so that you can import it back.</span></span>  
  
> [!IMPORTANT]  
>  <span data-ttu-id="54355-107">No puede importar o exportar diseños de informe integrados.</span><span class="sxs-lookup"><span data-stu-id="54355-107">You cannot import or export built-in report layouts.</span></span>  
  
### <a name="to-export-a-report-layout-to-a-file"></a><span data-ttu-id="54355-108">Para exportar un diseño de informe a un archivo</span><span class="sxs-lookup"><span data-stu-id="54355-108">To export a report layout to a file</span></span>  
  
1.  <span data-ttu-id="54355-109">Seleccione el icono ![Buscar página o informe](media/ui-search/search_small.png "icono Buscar página o informe"), escriba **Selección de diseño de informes** y, a continuación, seleccione el vínculo relacionado.</span><span class="sxs-lookup"><span data-stu-id="54355-109">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  
  
2.  <span data-ttu-id="54355-110">Seleccione la fila para el informe que contenga con el diseño personalizado que desee exportar y, a continuación, en la pestaña **Inicio**, en el grupo **Procesar**, elija **Diseños personalizados**.</span><span class="sxs-lookup"><span data-stu-id="54355-110">Select the row for the report that contains the custom report layout that you want to export, and then on the **Home** tab, in the **Process** group, choose **Custom Layouts**.</span></span>  
  
3.  <span data-ttu-id="54355-111">En la ventana **Diseños de informe**, seleccione el diseño de informe que desee exportar a un archivo y, a continuación, en la pestaña **Inicio**, en el grupo **Procesar**, elija **Exportar diseño**.</span><span class="sxs-lookup"><span data-stu-id="54355-111">In the **Report Layouts** window, select the report layout that you want to export to a file, and then on the **Home** tab, in the **Process** group, choose **Export Layout**.</span></span>  
  
4.  <span data-ttu-id="54355-112">En el cuadro de diálogo **Exportar archivo**, seleccione **Guardar** y, a continuación, guarde el archivo en una ubicación de su equipo o red.</span><span class="sxs-lookup"><span data-stu-id="54355-112">In the **Export File** dialog box, choose **Save**, and then save the file to a location on your computer or network.</span></span>  
  
### <a name="to-import-a-report-layout-file"></a><span data-ttu-id="54355-113">Para importar un archivo de diseño de informe</span><span class="sxs-lookup"><span data-stu-id="54355-113">To import a report layout file</span></span>  
  
1.  <span data-ttu-id="54355-114">Asegúrese de que el archivo correspondiente que define el diseño de informe está disponible en el equipo o en la red.</span><span class="sxs-lookup"><span data-stu-id="54355-114">Make sure that the relevant file that defines the report layout is available on your computer or network.</span></span>  
  
     <span data-ttu-id="54355-115">Un archivo de diseño de informe de Word debe tener la extensión de tipo de archivo .docx.</span><span class="sxs-lookup"><span data-stu-id="54355-115">A Word report layout file must have the .docx file type extension.</span></span> <span data-ttu-id="54355-116">El archivo de diseño de informe de RDLC debe tener la extensión de tipo de archivo .rdlc o .rdl.</span><span class="sxs-lookup"><span data-stu-id="54355-116">An RDLC report layout file must have the .rdlc or .rdl file type extension.</span></span>  
  
2.  <span data-ttu-id="54355-117">Seleccione el icono ![Buscar página o informe](media/ui-search/search_small.png "icono Buscar página o informe"), escriba **Selección de diseño de informes** y, a continuación, seleccione el vínculo relacionado.</span><span class="sxs-lookup"><span data-stu-id="54355-117">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  
  
3.  <span data-ttu-id="54355-118">Seleccione la fila para el informe al cual desee importar el diseño de informe y, a continuación, en la pestaña **Inicio**, en el grupo **Procesar**, elija **Diseños personalizados**.</span><span class="sxs-lookup"><span data-stu-id="54355-118">Select the row for the report to which you want to import the report layout, and then on the **Home** tab, in the **Process** group, choose **Custom Layouts**.</span></span>  
  
4.  <span data-ttu-id="54355-119">En la ventana **Diseño de informe**, seleccione el diseño de informe al que desee importar el archivo y, a continuación, en la pestaña **Inicio**, en el grupo **Procesar**, elija **Importar diseño**.</span><span class="sxs-lookup"><span data-stu-id="54355-119">In the **Report Layouts** window, select the report layout to which you want to import the file, and then on the **Home** tab, in the **Process** group, choose **Import Layout**.</span></span>  
  
5.  <span data-ttu-id="54355-120">En el cuadro de diálogo **Importar**, seleccione el documento que define el diseño de informe y, a continuación, elija **Abrir**.</span><span class="sxs-lookup"><span data-stu-id="54355-120">In the **Import** dialog box, select the document that defines the report layout, and then choose **Open**.</span></span>  
  
 <span data-ttu-id="54355-121">El diseño de informe original personalizado se reemplaza con el diseño de informe importado.</span><span class="sxs-lookup"><span data-stu-id="54355-121">The original custom report layout is replaced with the imported report layout.</span></span>
