# <a name="attaching-documents-to-master-data"></a>Adjuntar documentos a los datos maestros
Puede asociar todas las imágenes, documentos o imágenes a productos, a clientes, proveedores, a proyectos y recursos. Los archivos adjuntos también se adjuntarán a los documentos de compra y venta para ese registro principal.
