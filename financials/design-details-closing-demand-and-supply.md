# <a name="design-details-closing-demand-and-supply"></a><span data-ttu-id="26c39-103">Detalles de diseño: Cierre de aprovisionamiento y demanda</span><span class="sxs-lookup"><span data-stu-id="26c39-103">Design Details: Closing Demand and Supply</span></span>
<span data-ttu-id="26c39-104">Cuando se han realizado los procedimientos de equilibrado del suministro, existen tres situaciones finales posibles:</span><span class="sxs-lookup"><span data-stu-id="26c39-104">When the supply balancing procedures have been performed, there are three possible end situations:</span></span>  
  
* <span data-ttu-id="26c39-105">Se han cumplido la cantidad necesaria y la fecha de los eventos de demanda, y se puede cerrar la planificación de ellos.</span><span class="sxs-lookup"><span data-stu-id="26c39-105">The required quantity and date of the demand events have been met and the planning for them can be closed.</span></span> <span data-ttu-id="26c39-106">El evento de suministro aún está abierto y puede cubrir la demanda siguiente, por lo que el procedimiento de contrapartida puede empezar con el evento de suministro actual y la demanda siguiente.</span><span class="sxs-lookup"><span data-stu-id="26c39-106">The supply event is still open and may be able to cover the next demand, so the balancing procedure can start over with the current supply event and the next demand.</span></span>  
* <span data-ttu-id="26c39-107">El pedido de suministro no se puede modificar para cubrir toda la demanda.</span><span class="sxs-lookup"><span data-stu-id="26c39-107">The supply order cannot be modified to cover all of the demand.</span></span> <span data-ttu-id="26c39-108">El evento de demanda sigue abierto, con una cantidad sin cubrir que se puede cubrir mediante el evento de suministro siguiente.</span><span class="sxs-lookup"><span data-stu-id="26c39-108">The demand event is still open, with some uncovered quantity that may be covered by the next supply event.</span></span> <span data-ttu-id="26c39-109">Por lo tanto, se cierra el evento de suministro, por lo que el equilibrado puede empezar con la demanda actual y el siguiente evento de suministro.</span><span class="sxs-lookup"><span data-stu-id="26c39-109">Thus the current supply event is closed, so the balancing act can start over with the current demand and the next supply event.</span></span>  
* <span data-ttu-id="26c39-110">Se ha cubierto toda la demanda; no hay demanda subsiguiente (o no ha habido demanda en absoluto).</span><span class="sxs-lookup"><span data-stu-id="26c39-110">All of the demand has been covered; there is no subsequent demand (or there has been no demand at all).</span></span> <span data-ttu-id="26c39-111">Si hay algún aprovisionamiento de excedente, se puede reducir (o cancelarse) y después cerrar.</span><span class="sxs-lookup"><span data-stu-id="26c39-111">If there is any surplus supply, it may be decreased (or canceled) and then closed.</span></span> <span data-ttu-id="26c39-112">Es posible que existan eventos de suministro adicionales más adelante en la cadena y también se deben cancelar.</span><span class="sxs-lookup"><span data-stu-id="26c39-112">It is possible that additional supply events exist further along in the chain, and they should also be canceled.</span></span>  
  
<span data-ttu-id="26c39-113">Por último, el sistema de planificación creará una conexión de seguimiento de pedidos entre el suministro y la demanda.</span><span class="sxs-lookup"><span data-stu-id="26c39-113">Last, the planning system will create an order tracking link between the supply and the demand.</span></span>  
  
## <a name="creating-the-planning-line-suggested-action"></a><span data-ttu-id="26c39-114">Crear la línea de planificación (acción sugerida)</span><span class="sxs-lookup"><span data-stu-id="26c39-114">Creating the Planning Line (Suggested Action)</span></span>  
<span data-ttu-id="26c39-115">Si se sugiere alguna acción (nueva, cambiar de cantidad, reprogramar, reprogramar y cambiar de cantidad o cancelar) para revisar el pedido de aprovisionamiento, el sistema de planificación crea una línea de planificación en la hoja de trabajo de planificación.</span><span class="sxs-lookup"><span data-stu-id="26c39-115">If any action – New, Change Quantity, Reschedule, Reschedule and Change Quantity, or Cancel – is suggested to revise the supply order, the planning system creates a planning line in the planning worksheet.</span></span> <span data-ttu-id="26c39-116">El seguimiento de pedidos obliga a la creación de la línea de planificación no solo cuando se cierra el evento de aprovisionamiento, sino también si se cierra el evento de demanda, aunque el evento de aprovisionamiento esté aún abierto y puede estar sujeto a los cambios adicionales cuando se procesa el evento de demanda siguiente.</span><span class="sxs-lookup"><span data-stu-id="26c39-116">Due to order tracking, the planning line is created not only when the supply event is closed, but also if the demand event is closed, even though the supply event is still open and may be subject to additional changes when the next demand event is processed.</span></span> <span data-ttu-id="26c39-117">Esto significa que la línea de planificación se puede modificar de nuevo cuando se crea por primera vez.</span><span class="sxs-lookup"><span data-stu-id="26c39-117">This means that when first created, the planning line may be changed again.</span></span>  
  
<span data-ttu-id="26c39-118">Para minimizar el acceso de base de datos al manipular las órdenes de producción, la línea de planificación se puede mantener en tres niveles, a la vez que se intenta realizar el nivel de mantenimiento menos exigente:</span><span class="sxs-lookup"><span data-stu-id="26c39-118">To minimize database access when handling production orders, the planning line can be maintained in three levels, while aiming to perform the least demanding maintenance level:</span></span>  
  
* <span data-ttu-id="26c39-119">Crear solo la línea de planificación con la fecha de vencimiento y la cantidad actuales, pero sin la ruta y los componentes.</span><span class="sxs-lookup"><span data-stu-id="26c39-119">Create only the planning line with the current due date and quantity but without the routing and components.</span></span>  
* <span data-ttu-id="26c39-120">Incluir ruta: la ruta planificada se diseña incluyendo el cálculo de las fechas y las horas de inicio y fin.</span><span class="sxs-lookup"><span data-stu-id="26c39-120">Include routing: the planned routing is laid out including calculation of starting and ending dates and times.</span></span> <span data-ttu-id="26c39-121">Esto exige muchos accesos a la base de datos.</span><span class="sxs-lookup"><span data-stu-id="26c39-121">This is demanding in terms of database accesses.</span></span> <span data-ttu-id="26c39-122">Para determinar las fechas finales y de vencimiento, es posible que sea necesario calcularlo aunque el evento de suministro no se haya cerrado (en el caso de la programación hacia delante).</span><span class="sxs-lookup"><span data-stu-id="26c39-122">To determine the ending and due dates, it may be necessary to calculate this even if the supply event has not been closed (in the case of forward scheduling).</span></span>  
* <span data-ttu-id="26c39-123">Incluir despliegue de L.M.: esto puede esperar a justo antes de cerrar el evento de aprovisionamiento.</span><span class="sxs-lookup"><span data-stu-id="26c39-123">Include BOM explosion: this can wait until just before the supply event is closed.</span></span>  
  
<span data-ttu-id="26c39-124">De este modo concluyen las descripciones de cómo el sistema de planificación carga, da prioridad y equilibra la demanda y el suministro.</span><span class="sxs-lookup"><span data-stu-id="26c39-124">This concludes the descriptions of how demand and supply is loaded, prioritized, and balanced by the planning system.</span></span> <span data-ttu-id="26c39-125">En integración con esta actividad de planificación de aprovisionamientos, el programa debe garantizar que el nivel de inventario requerido de cada producto planificado se mantiene según sus directivas de reaprovisionamiento.</span><span class="sxs-lookup"><span data-stu-id="26c39-125">In integration with this supply planning activity, the system must ensure that the required inventory level of each planned item is maintained according to its reordering policies.</span></span>
