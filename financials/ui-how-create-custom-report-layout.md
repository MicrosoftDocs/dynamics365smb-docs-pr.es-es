## <a name="to-create-a-custom-layout"></a><span data-ttu-id="e1c3b-114">Para crear un diseño personalizado</span><span class="sxs-lookup"><span data-stu-id="e1c3b-114">To create a custom layout</span></span>
1. <span data-ttu-id="e1c3b-115">Seleccione el icono ![Buscar página o informe](media/ui-search/search_small.png "Search for Page or Report icon"), escriba **Selección de diseño de informes** y, a continuación, seleccione el vínculo relacionado.</span><span class="sxs-lookup"><span data-stu-id="e1c3b-115">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  
   <span data-ttu-id="e1c3b-116">La ventana **Selección de diseño de informes** muestra todos los informes disponibles en la empresa especificada en el campo Empresa en la parte superior de la ventana.</span><span class="sxs-lookup"><span data-stu-id="e1c3b-116">The** Report Layout Selection** window lists all the reports that are available in the company that is specified in the Company field at the top of the window.</span></span>
2. <span data-ttu-id="e1c3b-117">Establezca el campo **Empresa** en la empresa en la que desea crear el diseño del informe.</span><span class="sxs-lookup"><span data-stu-id="e1c3b-117">Set the **Company** field to the company in which you want to create the report layout.</span></span>
3. <span data-ttu-id="e1c3b-118">Seleccione la fila para el informe para el cual desee crear el diseño y, a continuación, elija **Diseños personalizados**.</span><span class="sxs-lookup"><span data-stu-id="e1c3b-118">Select the row for the report that you want to create the layout for, and then choose **Custom Layouts**.</span></span>

4. <span data-ttu-id="e1c3b-120">Si desea crear una copia de un diseño personalizado existente, seleccione el diseño personalizado existente en la lista y, a continuación, elija **Copiar**.</span><span class="sxs-lookup"><span data-stu-id="e1c3b-120">If you want to create a copy of an existing custom layout, select the existing custom layout in the list, and then choose **Copy**.</span></span>  
   <span data-ttu-id="e1c3b-121">La copia del diseño personalizado aparece en la ventana **Diseños de informe personalizados** e incluye las palabras Copia de en el campo Descripción.</span><span class="sxs-lookup"><span data-stu-id="e1c3b-121">The copy of the custom layout appears in the **Custom Report Layouts** window and has the words Copy of in the Description field.</span></span>
5. <span data-ttu-id="e1c3b-122">Si desea añadir un nuevo diseño personalizado que se base en un diseño integrado, haga lo siguiente:</span><span class="sxs-lookup"><span data-stu-id="e1c3b-122">If you want to add a new custom layout that is based on a built-in layout, do the following:</span></span>  
   1. <span data-ttu-id="e1c3b-123">Elija **Nuevo**.</span><span class="sxs-lookup"><span data-stu-id="e1c3b-123">Choose **New**.</span></span> <span data-ttu-id="e1c3b-124">Aparece la ventana **Insertar diseñado integrado para un informe**.</span><span class="sxs-lookup"><span data-stu-id="e1c3b-124">The **Insert Built-in Layout for a Report** window appears.</span></span> <span data-ttu-id="e1c3b-125">Los campos de **Id.** y **Nombre** se rellenan automáticamente.</span><span class="sxs-lookup"><span data-stu-id="e1c3b-125">The **ID** and **Name** fields are automatically filled in.</span></span>
   2. <span data-ttu-id="e1c3b-126">Para agregar un tipo de diseño de informe de Word personalizado, seleccione la casilla **Insertar diseño de Word**.</span><span class="sxs-lookup"><span data-stu-id="e1c3b-126">To add a custom Word report layout type, then select the **Insert Word Layout** check box.</span></span>
   3. <span data-ttu-id="e1c3b-127">Para agregar un tipo de diseño de informe de RDLC personalizado, seleccione la casilla **Insertar diseño de RDLC**.</span><span class="sxs-lookup"><span data-stu-id="e1c3b-127">To add a custom RDLC report layout type, then select the **Insert RDLC Layout** check box.</span></span>
   4. <span data-ttu-id="e1c3b-128">Elija el botón **Aceptar**.</span><span class="sxs-lookup"><span data-stu-id="e1c3b-128">Choose the **OK** button.</span></span>  
      <span data-ttu-id="e1c3b-129">Los nuevos diseños personalizados aparecen en la ventana **Diseños de informe personalizados**.</span><span class="sxs-lookup"><span data-stu-id="e1c3b-129">The new custom layouts appear in the **Custom Report Layouts** window.</span></span> <span data-ttu-id="e1c3b-130">Si un diseño nuevo se basa en un diseño integrado, tendrá las palabras **Copia de un diseño integrado** en el campo **Descripción**.</span><span class="sxs-lookup"><span data-stu-id="e1c3b-130">If a new layout is based on a built-in layout, then it has the words **Copy of a Built-in Layout** in the **Description** field.</span></span> <span data-ttu-id="e1c3b-131">Si no había diseño integrado para el informe, el nuevo diseño tendrá las palabras **Nuevo diseño** en el campo **Descripción** para indicar que el diseño personalizado está en blanco.</span><span class="sxs-lookup"><span data-stu-id="e1c3b-131">If there was no built-in layout for the report, then the new layout has the words **New Layout** in the **Description** field, which indicates that custom layout is blank.</span></span>
6. <span data-ttu-id="e1c3b-132">De forma predeterminada, el campo **Nombre de la empresa** está en blanco, lo que significa que el diseño personalizado estará disponible para el informe en todas las empresas.</span><span class="sxs-lookup"><span data-stu-id="e1c3b-132">By default, the **Company Name** field is blank, which means that the custom layout will be available for the report in all companies.</span></span> <span data-ttu-id="e1c3b-133">Para que el diseño personalizado esté disponible en una empresa específica solo, elija **Editar** y, a continuación, establezca el campo **Nombre de la empresa** en la empresa que quiera.</span><span class="sxs-lookup"><span data-stu-id="e1c3b-133">To make the custom layout available in a specific company only, choose **Edit**, and then set the **Company Name** field to the company that you want.</span></span>
