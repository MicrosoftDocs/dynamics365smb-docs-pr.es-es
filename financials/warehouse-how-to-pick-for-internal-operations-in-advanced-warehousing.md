## <a name="to-pick-components-using-the-pick-worksheet"></a><span data-ttu-id="79438-122">Para realizar el picking de componentes desde la hoja de trabajo de picking</span><span class="sxs-lookup"><span data-stu-id="79438-122">To pick components using the pick worksheet</span></span>  
1.  <span data-ttu-id="79438-123">Seleccione el icono ![Buscar página o informe](media/ui-search/search_small.png "icono Buscar página o informe"), escriba **Hoja trabajo picking** y, a continuación, seleccione el vínculo relacionado.</span><span class="sxs-lookup"><span data-stu-id="79438-123">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Pick Worksheet**, and then choose the related link.</span></span>

3.  <span data-ttu-id="79438-125">Explore las líneas, ordénelas para asegurar un picking más eficaz y combínelas con otras líneas de hoja de trabajo si es necesario para ahorrar tiempo a los empleados.</span><span class="sxs-lookup"><span data-stu-id="79438-125">Inspect the lines, sort them to ensure an efficient picking round, and combine them with other worksheet lines if necessary to make best use of employee time.</span></span>

6.  <span data-ttu-id="79438-128">Elija el botón **Aceptar**.</span><span class="sxs-lookup"><span data-stu-id="79438-128">Choose the **OK** button.</span></span> <span data-ttu-id="79438-129">Los documentos de picking de almacén se crean con las líneas de picking para cada componente necesario en la operación interna.</span><span class="sxs-lookup"><span data-stu-id="79438-129">Warehouse pick documents are created with pick lines for each component that is required in the internal operation.</span></span>  

<span data-ttu-id="79438-130">Si el área de operaciones internas, como un suelo de departamento de producción, se ha configurado con una ubicación predeterminada para la colocación de los componentes utilizados en la operación,ese código de ubicación se insertará en las líneas del lugar en el documento de picking de almacén para indicar a los trabajadores del almacén dónde ubicar los artículos.</span><span class="sxs-lookup"><span data-stu-id="79438-130">If the internal operation area, such as a production shop floor, is set up with a default bin for placement of components to be used in the operation, then that bin code is inserted in the Place lines on the warehouse pick document to instruct warehouse workers where to place the items.</span></span> <span data-ttu-id="79438-131">Para obtener más información, consulte el campo **Código de ubicación para producción** o **Código de ubicación para ensamblado**.</span><span class="sxs-lookup"><span data-stu-id="79438-131">For more information, see the **To-Production Bin Code** or the **To-Assembly Bin Code** field.</span></span>
