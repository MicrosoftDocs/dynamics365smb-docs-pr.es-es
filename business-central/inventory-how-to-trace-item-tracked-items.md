<span data-ttu-id="ad54b-106">Puede ver donde se ha utilizado un producto marcado para seguimiento, incluso cómo y dónde se recibió o se produjo, transfirió, vendió, consumió o devolvió.</span><span class="sxs-lookup"><span data-stu-id="ad54b-106">You can see where an item-tracked item was used, including how and when it was received or produced, transferred, sold, consumed, or returned.</span></span> <span data-ttu-id="ad54b-107">También puede encontrar todas las instancias actuales de un número de serie o de lote específico en la base de datos.</span><span class="sxs-lookup"><span data-stu-id="ad54b-107">You can also find all current instances of a specific serial or lot number in the database.</span></span> <span data-ttu-id="ad54b-108">Puede hacerlo con las características Seguimiento de producto y [Buscar movimientos](ui-find-entries.md).</span><span class="sxs-lookup"><span data-stu-id="ad54b-108">You do this by using the Item Tracing and the [Find Entries](ui-find-entries.md) features.</span></span>
