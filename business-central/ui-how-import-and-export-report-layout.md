>  <span data-ttu-id="88793-107">No puede importar o exportar diseños de informe integrados.</span><span class="sxs-lookup"><span data-stu-id="88793-107">You cannot import or export built-in report layouts.</span></span>  

### <a name="to-export-a-report-layout-to-a-file"></a><span data-ttu-id="88793-108">Para exportar un diseño de informe a un archivo</span><span class="sxs-lookup"><span data-stu-id="88793-108">To export a report layout to a file</span></span>  

1.  <span data-ttu-id="88793-109">Elija el icono ![Bombilla que abre la función Dígame](media/ui-search/search_small.png "Dígame qué desea hacer"), escriba **Selección de diseño de informes** y luego elija el enlace relacionado.</span><span class="sxs-lookup"><span data-stu-id="88793-109">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  

2.  <span data-ttu-id="88793-110">Seleccione la fila para el informe que contiene el diseño de informe personalizado que desea exportar y, a continuación, elija la acción **Diseños personalizados**.</span><span class="sxs-lookup"><span data-stu-id="88793-110">Select the row for the report that contains the custom report layout that you want to export, and then choose the **Custom Layouts** action.</span></span>  

3.  <span data-ttu-id="88793-111">En la página **Diseños de informe**, seleccione el diseño de informe que desee exportar a un archivo y, a continuación, elija la acción **Exportar diseño**.</span><span class="sxs-lookup"><span data-stu-id="88793-111">On the **Report Layouts** page, select the report layout that you want to export to a file, and then choose the **Export Layout** action.</span></span>  

4.  <span data-ttu-id="88793-112">En el cuadro de diálogo **Exportar archivo**, elija el botón **Guardar** y, a continuación, guarde el archivo en una ubicación de su equipo o red.</span><span class="sxs-lookup"><span data-stu-id="88793-112">In the **Export File** dialog box, choose the **Save** button, and then save the file to a location on your computer or network.</span></span>  

### <a name="to-import-a-report-layout-file"></a><span data-ttu-id="88793-113">Para importar un archivo de diseño de informe</span><span class="sxs-lookup"><span data-stu-id="88793-113">To import a report layout file</span></span>  

1.  <span data-ttu-id="88793-114">Asegúrese de que el archivo correspondiente que define el diseño de informe está disponible en el equipo o en la red.</span><span class="sxs-lookup"><span data-stu-id="88793-114">Make sure that the relevant file that defines the report layout is available on your computer or network.</span></span>  

     <span data-ttu-id="88793-115">Un archivo de diseño de informe de Word debe tener la extensión de tipo de archivo .docx.</span><span class="sxs-lookup"><span data-stu-id="88793-115">A Word report layout file must have the .docx file type extension.</span></span> <span data-ttu-id="88793-116">El archivo de diseño de informe de RDLC debe tener la extensión de tipo de archivo .rdlc o .rdl.</span><span class="sxs-lookup"><span data-stu-id="88793-116">An RDLC report layout file must have the .rdlc or .rdl file type extension.</span></span>  

2.  <span data-ttu-id="88793-117">Elija el icono ![Bombilla que abre la función Dígame](media/ui-search/search_small.png "Dígame qué desea hacer"), escriba **Selección de diseño de informes** y luego elija el enlace relacionado.</span><span class="sxs-lookup"><span data-stu-id="88793-117">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  

3.  <span data-ttu-id="88793-118">Seleccione la fila para el informe para el que desee importar el diseño de informe y, a continuación, elija la acción **Diseños personalizados**.</span><span class="sxs-lookup"><span data-stu-id="88793-118">Select the row for the report to which you want to import the report layout, and then choose the **Custom Layouts** action.</span></span>  

4.  <span data-ttu-id="88793-119">En la página **Diseños de informe**, seleccione el diseño de informe en el que desee importar el archivo y, a continuación, elija la acción **Importar diseño**.</span><span class="sxs-lookup"><span data-stu-id="88793-119">On the **Report Layouts** page, select the report layout to which you want to import the file, and then choose the **Import Layout** action.</span></span>  

5.  <span data-ttu-id="88793-120">En el cuadro de diálogo **Importar**, seleccione el documento que define el diseño de informe y, a continuación, elija el botón **Abrir**.</span><span class="sxs-lookup"><span data-stu-id="88793-120">In the **Import** dialog box, select the document that defines the report layout, and then choose the **Open** button.</span></span>  

 <span data-ttu-id="88793-121">El diseño de informe original personalizado se reemplaza con el diseño de informe importado.</span><span class="sxs-lookup"><span data-stu-id="88793-121">The original custom report layout is replaced with the imported report layout.</span></span>
