 <span data-ttu-id="820b9-106">Como para cualquier otro tipo de artículo, también puede crear pedidos abiertos de venta para los artículos de ensamblado personalizados antes de realizar el pedido de ventas real según el acuerdo del pedido abierto.</span><span class="sxs-lookup"><span data-stu-id="820b9-106">As with any other type of item, you can also create blanket sales orders for customized assembly items before periodically making the actual sales orders according to the blanket order agreement.</span></span> <span data-ttu-id="820b9-107">Este proceso implica varios pasos adicionales cuando se compara con la creación de pedidos de ventas abiertos periódicos y utiliza una variación del pedido de ensamblado vinculado, que es un pedido de ensamblado abierto.</span><span class="sxs-lookup"><span data-stu-id="820b9-107">This process involves several extra steps when you compare it to creating a regular blanket sales order, and it uses a variation of a linked assembly order, which is a blanket assembly order.</span></span>

>  <span data-ttu-id="820b9-108">Al igual que todos los pedidos abiertos, las cantidades de los pedidos abiertos de ensamblado solo son previsiones y no están operativas hasta que se convierten en pedidos de ensamblado reales.</span><span class="sxs-lookup"><span data-stu-id="820b9-108">Like all blanket orders, quantities on assembly blanket orders are only forecasts and are not operational until they are converted to actual assembly orders.</span></span> <span data-ttu-id="820b9-109">Por tanto, la funcionalidad del pedido, como el cálculo de disponibilidad, la reserva y el seguimiento de artículos no están activos en los pedidos de ensamblado abiertos.</span><span class="sxs-lookup"><span data-stu-id="820b9-109">Therefore, order functionality, such as availability calculation, reservation, and item tracking, is not active on blanket assembly orders.</span></span>  

## <a name="to-create-a-blanket-assembly-order-for-an-assemble-to-order-item"></a><span data-ttu-id="820b9-110">Crear un pedido de ensamblado abierto para un artículo de ensamblado para pedido</span><span class="sxs-lookup"><span data-stu-id="820b9-110">To create a blanket assembly order for an assemble\-to\-order item</span></span>  
1. <span data-ttu-id="820b9-111">Elija el icono ![Bombilla que abre la función Dígame](media/ui-search/search_small.png "Dígame qué desea hacer"), escriba **Pedidos de venta abiertos** y luego elija el enlace relacionado.</span><span class="sxs-lookup"><span data-stu-id="820b9-111">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Blanket Sales Orders**, and then choose the related link.</span></span>  
2. <span data-ttu-id="820b9-112">Cree un nuevo pedido de ventas abierto con una línea para un artículo de ensamblado.</span><span class="sxs-lookup"><span data-stu-id="820b9-112">Create a new blanket sales order with one line for an assembly item.</span></span> <span data-ttu-id="820b9-113">Consulte [Crear pedidos abiertos de venta](sales-how-to-create-blanket-sales-orders.md) para obtener más información.</span><span class="sxs-lookup"><span data-stu-id="820b9-113">For more information, see [Create Blanket Sales Orders](sales-how-to-create-blanket-sales-orders.md).</span></span>  
3. <span data-ttu-id="820b9-114">En el campo **Cantidad a ensamblar para pedido** en la línea del pedido de ensamblado abierto, escriba la cantidad total.</span><span class="sxs-lookup"><span data-stu-id="820b9-114">In the **Qty. to Assemble to Order** field on the blanket assembly order line, enter the full quantity.</span></span>

    > [!NOTE]  
    >  <span data-ttu-id="820b9-115">No debe crear los acuerdos de pedido abierto para una cantidad parcial.</span><span class="sxs-lookup"><span data-stu-id="820b9-115">You should not create blanket order agreements for a partial quantity.</span></span> <span data-ttu-id="820b9-116">Por tanto, deberá especificar la misma cantidad que haya especificado en el campo **Cantidad** de la línea de pedido de ventas abierto.</span><span class="sxs-lookup"><span data-stu-id="820b9-116">Therefore, you must enter the same quantity that you entered in the **Quantity** field on the blanket sales order line.</span></span>  

4. <span data-ttu-id="820b9-117">Elija la acción **Ensamblar para pedido** y, a continuación, **Ensamblar para líneas de pedido**.</span><span class="sxs-lookup"><span data-stu-id="820b9-117">Choose the **Assemble to Order** action, and then choose the **Assemble-to-Order Lines** action.</span></span> <span data-ttu-id="820b9-118">También puede elegir el campo **Cdad. al ensamblar para pedido** en la línea.</span><span class="sxs-lookup"><span data-stu-id="820b9-118">Alternatively, choose the **Qty. to Assemble to Order)** field on the line.</span></span>  
5. <span data-ttu-id="820b9-119">En la página **Líneas de ensamblado para pedido**, revise o modifique las líneas de pedido de ensamblado de acuerdo con el acuerdo de pedido abierto que haya suscrito con el cliente.</span><span class="sxs-lookup"><span data-stu-id="820b9-119">On the **Assemble-to-Order Lines** page, review or modify the assembly order lines according to the blanket order agreement that you have made with the customer.</span></span> <span data-ttu-id="820b9-120">Si desea ver más información, seleccione la acción **Mostrar documento** para abrir el pedido de ensamblado abierto completo.</span><span class="sxs-lookup"><span data-stu-id="820b9-120">If you want to view more information, choose the **Show Document** action to open the complete blanket assembly order.</span></span> <span data-ttu-id="820b9-121">No puede modificar el contenido de la mayoría de los campos, y no puede realizar ningún registro.</span><span class="sxs-lookup"><span data-stu-id="820b9-121">You cannot change the contents of most fields, and you cannot post.</span></span>  
6. <span data-ttu-id="820b9-122">Cuando se hayan ajustado las líneas del pedido de ensamblado según el acuerdo de pedido abierto, cierre la página **Líneas de ensamblado para pedido** para volver a la página **Pedido abierto de ventas**.</span><span class="sxs-lookup"><span data-stu-id="820b9-122">When you have adjusted the assembly order lines according to the blanket order agreement, close the **Assemble-to-Order Lines** page to return to the **Blanket Sales Order** page.</span></span>  
7. <span data-ttu-id="820b9-123">Cuando el cliente solicita crear un pedido de venta basado en el pedido de ventas abierto acordado, crea un pedido de venta para los artículos de ensamblado acordados.</span><span class="sxs-lookup"><span data-stu-id="820b9-123">When the customer requests to create a sales order based on the agreed blanket sales order, create a sales order for the agreed assembly item or items.</span></span> <span data-ttu-id="820b9-124">Consulte [Crear pedidos abiertos de venta](sales-how-to-create-blanket-sales-orders.md) para obtener más información.</span><span class="sxs-lookup"><span data-stu-id="820b9-124">For more information, see [Create Blanket Sales Orders](sales-how-to-create-blanket-sales-orders.md).</span></span>

<span data-ttu-id="820b9-125">El pedido de ensamblado abierto vinculado y las personalizaciones se vinculan a ese nuevo pedido de ventas para preparar el ensamblado de los artículos que se van a vender.</span><span class="sxs-lookup"><span data-stu-id="820b9-125">The linked blanket assembly order and any customizations are linked to that new sales order to prepare for assembly of the item or items to be sold.</span></span>
