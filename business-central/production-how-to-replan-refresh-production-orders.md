# <a name="replan-or-refresh-production-orders-directly"></a><span data-ttu-id="56f8a-103">Replanificar o actualizar órdenes de producción directamente</span><span class="sxs-lookup"><span data-stu-id="56f8a-103">Replan or Refresh Production Orders Directly</span></span>
<span data-ttu-id="56f8a-104">Se utiliza normalmente la función **Replanificar** después de haber agregado o cambiado componentes que conforman órdenes de producción subyacentes.</span><span class="sxs-lookup"><span data-stu-id="56f8a-104">The **Replan** function on production orders is typically used after you have added or changed components that constitute underlying production orders.</span></span> <span data-ttu-id="56f8a-105">La función de planificación calcula los cambios realizados en las líneas de componentes y rutas e incluye a los productos de los niveles de L.M. de producción inferiores, para los que puede crear nuevas órdenes de producción.</span><span class="sxs-lookup"><span data-stu-id="56f8a-105">The function calculates changes made to components and routings lines, and it includes items on lower production BOM levels for which it may generate new production orders.</span></span>  

<span data-ttu-id="56f8a-106">De acuerdo con los cambios realizados en las líneas de componentes y rutas, la función de replanificación calcula y planifica la nueva demanda de la orden de producción.</span><span class="sxs-lookup"><span data-stu-id="56f8a-106">Based on the changes you have made to the components and routing lines, the Replan function calculates and plans for any new demand for the production order.</span></span>  

<span data-ttu-id="56f8a-107">La función **Actualizar** en órdenes de producción se utiliza normalmente después de que se haya hecho una de las siguientes opciones:</span><span class="sxs-lookup"><span data-stu-id="56f8a-107">The **Refresh** function on production orders is typically used after you have done one of the following:</span></span>

- <span data-ttu-id="56f8a-108">Se crea manualmente una cabecera de orden de producción para calcular y crear por primera vez los datos de las líneas.</span><span class="sxs-lookup"><span data-stu-id="56f8a-108">Created a production order header manually to calculate and create line data for the first time.</span></span>
- <span data-ttu-id="56f8a-109">Se realizan cambios en la cabecera de la orden de producción para volver a calcular los datos de las líneas.</span><span class="sxs-lookup"><span data-stu-id="56f8a-109">Made changes to the production order header to recalculate all the line data.</span></span>

<span data-ttu-id="56f8a-110">La función de actualizar calcula los cambios realizados en una cabecera de orden de producción sin que resulten afectados los niveles de L.M. de producción.</span><span class="sxs-lookup"><span data-stu-id="56f8a-110">The Refresh function calculates changes made to a production order header and does not involve production BOM levels.</span></span> <span data-ttu-id="56f8a-111">La función calcula e inicia los valores de las líneas de componentes y de ruta basada en los datos maestros definidos en la L.M. de producción asignada y la ruta, de acuerdo con la cantidad a solicitar y la fecha de vencimiento de la cabecera de la orden de producción.</span><span class="sxs-lookup"><span data-stu-id="56f8a-111">The function calculates and initiates the values of the component lines and routing lines based on the master data defined in the assigned production BOM and routing, according to the order quantity and due date on the production order’s header.</span></span>

<span data-ttu-id="56f8a-112">Puede insertar las líneas de la orden de producción manualmente o utilizar la función que calcula las líneas desde la cabecera.</span><span class="sxs-lookup"><span data-stu-id="56f8a-112">You can either insert the production order lines manually or use the function that calculates the production order lines from the header.</span></span>  

> [!NOTE]
> <span data-ttu-id="56f8a-113">Si utiliza la función Actualizar para recalcular las líneas de orden de producción, se eliminan las líneas antiguas y se calculan las nuevas.</span><span class="sxs-lookup"><span data-stu-id="56f8a-113">If you use the Refresh function to recalculate production order lines, the old production order lines are deleted and new lines are calculated.</span></span>
