## <a name="to-move-components-to-an-operation-area-in-basic-warehouse-configurations"></a><span data-ttu-id="9e901-123">Para mover componentes a un área de operaciones en configuraciones básicas de almacén</span><span class="sxs-lookup"><span data-stu-id="9e901-123">To move components to an operation area in basic warehouse configurations</span></span>  
1.  <span data-ttu-id="9e901-124">Elija el icono ![Bombilla que abre la función Dígame](media/ui-search/search_small.png "Dígame qué desea hacer"), escriba **Movimiento de inventario** y elija el enlace relevante.</span><span class="sxs-lookup"><span data-stu-id="9e901-124">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Inventory Movement**, and choose the relevant link.</span></span>  
2.  <span data-ttu-id="9e901-125">En la ficha desplegable **General**, rellene el campo **Nº**</span><span class="sxs-lookup"><span data-stu-id="9e901-125">On the **General** FastTab, fill in the **No.**</span></span> <span data-ttu-id="9e901-126">.</span><span class="sxs-lookup"><span data-stu-id="9e901-126">field.</span></span> <span data-ttu-id="9e901-127">Puede pulsar la tecla Entrar para seleccionar entre las series numéricas.</span><span class="sxs-lookup"><span data-stu-id="9e901-127">You can press the Enter key  to select from the number series.</span></span>  
3.  <span data-ttu-id="9e901-128">En el campo **Cód. almacén**, escriba la ubicación donde tiene lugar el movimiento.</span><span class="sxs-lookup"><span data-stu-id="9e901-128">In the **Location Code** field, enter the location where the movement occurs.</span></span>  
4.  <span data-ttu-id="9e901-129">Seleccione la acción **Traer doc. origen**.</span><span class="sxs-lookup"><span data-stu-id="9e901-129">Choose the **Get Source Documents** action.</span></span> <span data-ttu-id="9e901-130">Alternativamente, rellene el campo **Documento origen** y, a continuación, elija el botón **AssistEdit** en el campo **Cód. procedencia mov.**</span><span class="sxs-lookup"><span data-stu-id="9e901-130">Alternatively, fill in the **Source Document** field, and then choose the **AssistEdit** button in the **Source No.** field.</span></span>  
5.  <span data-ttu-id="9e901-131">En la página **Documentos origen**, seleccione el pedido de ensamblado al que desea mover los componentes y, a continuación seleccione el botón **Aceptar**.</span><span class="sxs-lookup"><span data-stu-id="9e901-131">On the **Source Documents** page, select the assembly order that you want to move components for, and then choose the **OK** button.</span></span>  

    <span data-ttu-id="9e901-132">Para cada componente necesario que se pueda mover, se generan una línea Toma y una línea Plaza en la página **Movimientos de inventario**.</span><span class="sxs-lookup"><span data-stu-id="9e901-132">For each needed component that can be moved, one Take line and one Place line are generated on the **Inventory Movements** page.</span></span> <span data-ttu-id="9e901-133">Todos los campos excepto **Cdad. a manipular** se prellenan según las líneas del documento de origen.</span><span class="sxs-lookup"><span data-stu-id="9e901-133">All fields except the **Qty. to Handle** field are prefilled according to the source document lines.</span></span> <span data-ttu-id="9e901-134">El campo **Cdad. a manipular** se establece en cero hasta que se introduzca la cantidad que realmente ha movido.</span><span class="sxs-lookup"><span data-stu-id="9e901-134">The **Qty. to Handle** field is set to zero until you enter the quantity that you have actually moved.</span></span>  

    <span data-ttu-id="9e901-135">Puede modificar el código de ubicación en una línea de Toma pero sólo según la disponibilidad.</span><span class="sxs-lookup"><span data-stu-id="9e901-135">You can change the bin code on a Take line but only according to availability.</span></span> <span data-ttu-id="9e901-136">Si elige el botón **AssistEdit** en el campo **Cód. ubicación** en una línea Toma, se abre la página **Contenidos ubicación** y sólo muestra ubicaciones donde está disponible el componente.</span><span class="sxs-lookup"><span data-stu-id="9e901-136">If you choose the **AssistEdit** button in the **Bin Code** field on a Take line, then the **Bin Contents** page opens and only shows bins where the component is available.</span></span>  

    <span data-ttu-id="9e901-137">No puede modificar el código de ubicación en una línea Plaza.</span><span class="sxs-lookup"><span data-stu-id="9e901-137">You cannot change the bin code on a Place line.</span></span> <span data-ttu-id="9e901-138">Sólo se valida el código de ubicación que se define en la línea de componente del documento de origen.</span><span class="sxs-lookup"><span data-stu-id="9e901-138">Only the bin code that is defined on the component line of the source document is accepted.</span></span> <span data-ttu-id="9e901-139">Esto soporta el principio de que la función que solicita un componente, que es un trabajador de ensamblado en este procedimiento, sabe dónde debe situarse el componente.</span><span class="sxs-lookup"><span data-stu-id="9e901-139">This supports the principle that the role who requests a component, which is an assembly worker in this procedure, knows where the component must be placed.</span></span> <span data-ttu-id="9e901-140">Si desea situar los componentes en otra ubicación, primero deberá modificar el código de ubicación en la línea de componente y después volver a crear las líneas del movimiento de inventario.</span><span class="sxs-lookup"><span data-stu-id="9e901-140">If you want to place the components in a different bin, then you must first change the bin code on the component line and then re-create the inventory movement lines.</span></span>  
6.  <span data-ttu-id="9e901-141">En el campo **Cdad. a manipular**, introduzca la cantidad completa o parcial cantidad que ha movido realmente.</span><span class="sxs-lookup"><span data-stu-id="9e901-141">In the **Qty. to Handle** field, enter the full or partial quantity that you have actually moved.</span></span> <span data-ttu-id="9e901-142">El valor de Toma y las líneas de Plaza deben iguales.</span><span class="sxs-lookup"><span data-stu-id="9e901-142">The value on the Take and the Place lines must be the same.</span></span> <span data-ttu-id="9e901-143">Si no, no podrá registrar el movimiento.</span><span class="sxs-lookup"><span data-stu-id="9e901-143">Otherwise, you cannot register the movement.</span></span>  

    > [!NOTE]  
    >  <span data-ttu-id="9e901-144">Como en otras actividades de almacén, puede dividir la línea del apartado seleccionando **Acciones** y **Dividir línea**.</span><span class="sxs-lookup"><span data-stu-id="9e901-144">As in other warehouse activities, you can split the Place line by selecting the **Actions** action and then choosing the **Split Line** action.</span></span> <span data-ttu-id="9e901-145">En ese caso, la suma de las dos líneas divididas de Plaza debe ser igual a la cantidad en la línea de Toma.</span><span class="sxs-lookup"><span data-stu-id="9e901-145">In that case, the sum of the two split Place lines must equal the quantity on the Take line.</span></span>  

7.  <span data-ttu-id="9e901-146">Cuando vaya para registrar los movimientos realizados, seleccione la acción **Registro Invent. movimiento**.</span><span class="sxs-lookup"><span data-stu-id="9e901-146">When you are ready to register the movements that you have performed, choose the **Register Invt. Movement** action.</span></span>  

    <span data-ttu-id="9e901-147">Los movimientos de almacén se crean reflejando que los componentes ahora existen en las ubicaciones especificadas en las líneas del pedido de ensamblado.</span><span class="sxs-lookup"><span data-stu-id="9e901-147">Warehouse entries are created reflecting that the components now exist in the bins specified on the assembly order lines.</span></span>  

    > [!NOTE]  
    >  <span data-ttu-id="9e901-148">A diferencia de cuando se mueven los componentes con un picking de inventario, el consumo no se registra cuando se registra un movimiento de inventario.</span><span class="sxs-lookup"><span data-stu-id="9e901-148">Unlike when you move components with an inventory pick, consumption is not posted when you register an inventory movement.</span></span> <span data-ttu-id="9e901-149">Ese el paso puede realizarse por separado registrando la salida y el consumo del pedido de ensamblado.</span><span class="sxs-lookup"><span data-stu-id="9e901-149">That step must be performed separately by posting the assembly order output and consumption.</span></span> <span data-ttu-id="9e901-150">Para obtener más información, consulte Pedido de ensamblado.</span><span class="sxs-lookup"><span data-stu-id="9e901-150">For more information, see Assembly Order.</span></span>
