# <a name="create-contacts"></a><span data-ttu-id="d9900-103">Crear contactos</span><span class="sxs-lookup"><span data-stu-id="d9900-103">Create Contacts</span></span>
<span data-ttu-id="d9900-104">Puede crear un contacto por cada nueva empresa con la que se relacione, como puede ser un cliente, un proveedor, un posible cliente, un banco, un despacho de abogados, una consultora, etc.</span><span class="sxs-lookup"><span data-stu-id="d9900-104">You can create a contact for each new company you interact with, for example, a customer, vendor, prospective customer, bank, law firm, consultant, and so on.</span></span>

<span data-ttu-id="d9900-105">Existen dos formas de crear un contacto: desde cero o a partir de un cliente, proveedor o cuenta bancaria existente.</span><span class="sxs-lookup"><span data-stu-id="d9900-105">There are two ways to create a contact: from scratch or from an existing customer, vendor, or bank account.</span></span>

## <a name="create-a-company-contact-from-scratch"></a><span data-ttu-id="d9900-106">Crear un contacto de compañía desde cero</span><span class="sxs-lookup"><span data-stu-id="d9900-106">Create a company contact from scratch</span></span>
1. <span data-ttu-id="d9900-107">Elija el icono ![bombilla que abre la función Dígame](media/ui-search/search_small.png "Dígame que desea hacer"), escriba **Contactos** y luego elija el enlace relacionado.</span><span class="sxs-lookup"><span data-stu-id="d9900-107">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Contacts**, and then choose the related link.</span></span>
2. <span data-ttu-id="d9900-108">Seleccione la acción **Nuevo**.</span><span class="sxs-lookup"><span data-stu-id="d9900-108">Choose the **New** action.</span></span>
3. <span data-ttu-id="d9900-109">En el campo **N.º de campo** introduzca un número para el contacto.</span><span class="sxs-lookup"><span data-stu-id="d9900-109">In the **No. field**, enter a number for the contact.</span></span>

    <span data-ttu-id="d9900-110">Por otra parte, si configuró un número de serie para los contactos de la página **Configuración marketing**, puede pulsar Entrar para hacer que el sistema especifique el siguiente número de contacto disponible.</span><span class="sxs-lookup"><span data-stu-id="d9900-110">Alternatively, if you have set up a number series for contacts on the **Marketing Setup** page, you can press the Enter key to select the next available contact number.</span></span>  
4. <span data-ttu-id="d9900-111">Establezca **Tipo** en **Empresa**.</span><span class="sxs-lookup"><span data-stu-id="d9900-111">Set **Type** to **Company**.</span></span>
5. <span data-ttu-id="d9900-112">Rellene los demás campos según sea necesario.</span><span class="sxs-lookup"><span data-stu-id="d9900-112">Fill in the other fields as required.</span></span>

## <a name="to-create-a-company-contact-from-a-customer-vendor-or-bank-account"></a><span data-ttu-id="d9900-113">Para crear un contacto desde un cliente, proveedor o banco</span><span class="sxs-lookup"><span data-stu-id="d9900-113">To create a company contact from a customer, vendor, or bank account</span></span>
<span data-ttu-id="d9900-114">Si ya configuró un cierto número de clientes, proveedores o bancos, podrá crear contactos según los datos existentes.</span><span class="sxs-lookup"><span data-stu-id="d9900-114">If you have already set up a number of customers, vendors, and bank accounts, you can create contacts on the basis of the existing data.</span></span> <span data-ttu-id="d9900-115">Al crear un contacto de esta manera, la información de contacto se sincroniza con la información del cliente, proveedor o de la cuenta bancaria.</span><span class="sxs-lookup"><span data-stu-id="d9900-115">When you create a contact this way, the contact information is synchronized with the customer, vendor, or bank account information.</span></span>

> [!NOTE]  
>   <span data-ttu-id="d9900-116">Para poder crear empresas de contacto, es necesario especificar un código de relación de negocio para los clientes, proveedores y las cuentas bancarias en la página **Configuración de marketing**.</span><span class="sxs-lookup"><span data-stu-id="d9900-116">Before you can create contact companies this way, you must specify a business relation code for customers, vendors, and bank accounts on the **Marketing Setup** page.</span></span> <span data-ttu-id="d9900-117">Si va a crear contactos a partir de cuentas bancarias, también debe especificar números de serie para las cuentas bancarias en la página **Configuración de contabilidad**.</span><span class="sxs-lookup"><span data-stu-id="d9900-117">If you will be creating contacts from a bank accounts, you must also specify numbers series for bank accounts on the **General Ledger Setup** page.</span></span>

1. <span data-ttu-id="d9900-118">Elija el icono ![bombilla que abre la función Dígame](media/ui-search/search_small.png "Dígame que desea hacer"), especifique una de las siguientes opciones, según la ubicación desde la cual desee crear contactos y, a continuación, elija el vínculo relacionado.</span><span class="sxs-lookup"><span data-stu-id="d9900-118">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter one of the following, depending on from where you want to create contacts, and then choose the related link.</span></span>
   * <span data-ttu-id="d9900-119">**Crear Contactos desde Clientes**</span><span class="sxs-lookup"><span data-stu-id="d9900-119">**Create Contacts from Customers**</span></span>
   * <span data-ttu-id="d9900-120">**Crear contactos desde proveedores**</span><span class="sxs-lookup"><span data-stu-id="d9900-120">**Create Contacts from Vendors**</span></span>
   * <span data-ttu-id="d9900-121">**Crear contactos desde bancos**</span><span class="sxs-lookup"><span data-stu-id="d9900-121">**Create Contacts from Bank Accounts**</span></span>
2. <span data-ttu-id="d9900-122">En la página del proceso que se abre, establezca filtros en la sección **Cliente**, **Proveedor** o **Banco** si desea crear contactos solo desde determinados clientes, proveedores o bancos.</span><span class="sxs-lookup"><span data-stu-id="d9900-122">In the batch job page that opens, in the **Customer**, **Vendor**, or **Bank Account** section, set filters if you want to create contacts from specific customers, vendors, or bank accounts.</span></span>
3. <span data-ttu-id="d9900-123">Para crear contactos, seleccione **Aceptar**.</span><span class="sxs-lookup"><span data-stu-id="d9900-123">Choose the **OK** button to start creating contacts.</span></span>

    <span data-ttu-id="d9900-124">Los siguientes números de contacto de la serie se asignan a los nuevos contactos.</span><span class="sxs-lookup"><span data-stu-id="d9900-124">The next contact numbers in the number series are assigned to the new contacts.</span></span> <span data-ttu-id="d9900-125">El programa asigna la relación comercial para proveedores especificada en la página **Configuración marketing** a los contactos recién creados.</span><span class="sxs-lookup"><span data-stu-id="d9900-125">The business relation for vendors that is specified on the **Marketing Setup** page is assigned to the newly created contacts.</span></span>

> [!TIP]  
>   <span data-ttu-id="d9900-126">También puede crear un cliente, proveedor o una cuenta bancaria a partir de un contacto.</span><span class="sxs-lookup"><span data-stu-id="d9900-126">You can also create a customer, vendor, or bank account from a contact.</span></span> <span data-ttu-id="d9900-127">Para obtener más información, consulte [Crear un cliente, proveedor o cuenta bancaria a partir de un contacto](marketing-how-create-contacts-new-customers-vendors-bank-accounts.md).</span><span class="sxs-lookup"><span data-stu-id="d9900-127">For more information, see [Create a Customer, Vendor, or Bank Account From a Contact](marketing-how-create-contacts-new-customers-vendors-bank-accounts.md).</span></span>
