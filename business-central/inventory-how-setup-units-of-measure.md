# <a name="set-up-item-units-of-measure"></a><span data-ttu-id="cc3a7-103">Configurar unidades de medida de producto</span><span class="sxs-lookup"><span data-stu-id="cc3a7-103">Set Up Item Units of Measure</span></span>
<span data-ttu-id="cc3a7-104">Puede configurar varias unidades de medida para un producto de forma que pueda asignar las unidades de medida al producto para los fines siguientes:</span><span class="sxs-lookup"><span data-stu-id="cc3a7-104">You can set up multiple units of measure for an item so that you can assign units of measure to the item for the following purposes:</span></span>

- <span data-ttu-id="cc3a7-105">Asigne una unidad de medida base en la ficha de producto para definir cómo se almacena en el inventario y para utilizar como base de conversión para las unidades de medida alternas.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-105">Assign a base unit of measure on the item’s item card to define how it is stored in inventory and to serve as the conversion basis for alternate units of measure.</span></span>
- <span data-ttu-id="cc3a7-106">Asigne las unidades de medida alternas a los documentos de compra, producción o venta para especificar cuántas unidades de la unidad de medida base que se llevarán al mismo tiempo en dichos procesos.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-106">Assign alternate units of measure to purchase, production, or sales documents to specify how many units of the base unit of measure you handle at a time in those processes.</span></span> <span data-ttu-id="cc3a7-107">Por ejemplo, puede comprar el producto en palés y utilizar solo piezas sueltas en la producción.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-107">For example, you may buy the item on pallets and only use single pieces in your production.</span></span>

<span data-ttu-id="cc3a7-108">Si un producto se almacena en una unidad de medida pero se fabrica en otra, se crea una orden de producción que utiliza una unidad de medida de la sección de fabricación para calcular la cantidad correcta de componentes durante el trabajo por lotes **Actualizar orden producción**.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-108">If an item is stocked in one unit of measure but produced in another, a production order is created that uses a manufacturing batch unit of measure to calculate the correct quantity of the components during the **Refresh Production Order** batch job.</span></span> <span data-ttu-id="cc3a7-109">Un ejemplo del cálculo de una unidad de medida de la sección de fabricación es el caso en que un producto fabricado se almacena por piezas pero se produce en toneladas.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-109">An example of a manufacturing batch unit of measure calculation is when a manufactured item is stocked in pieces but produced in tons.</span></span> <span data-ttu-id="cc3a7-110">Para obtener más información, consulte [Trabajar con la unidad de medida de lote de fabricación](production-how-to-use-the-manufacturing-batch-unit-of-measure.md).</span><span class="sxs-lookup"><span data-stu-id="cc3a7-110">For more information, see [Work with Manufacturing Batch Units of Measure](production-how-to-use-the-manufacturing-batch-unit-of-measure.md).</span></span>

## <a name="to-set-up-a-unit-of-measure"></a><span data-ttu-id="cc3a7-111">Configurar una unidad de medida</span><span class="sxs-lookup"><span data-stu-id="cc3a7-111">To set up a unit of measure</span></span>
1. <span data-ttu-id="cc3a7-112">Elija el icono ![bombilla que abre la función Dígame](media/ui-search/search_small.png "Dígame que desea hacer"), escriba **Productos** y luego elija el enlace relacionado.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-112">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Items**, and then choose the related link.</span></span>
2. <span data-ttu-id="cc3a7-113">Abra la ficha del producto para el que desea configurar unidades de medida alternas.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-113">Open the card of the item for which you want to set up alternate units of measure.</span></span>
3. <span data-ttu-id="cc3a7-114">Elija la acción **Unidades medida**.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-114">Choose the **Units of Measure** action.</span></span> <span data-ttu-id="cc3a7-115">Se abre la ventana **Unidades medida producto**.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-115">The **Item Units of Measure** window opens.</span></span>
4. <span data-ttu-id="cc3a7-116">Si el campo **Unidad de medida base** en la ficha de producto se ha rellenado, dicha unidad de medida ya está configurada.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-116">If the **Base Unit of Measure** field on the item card is filled, then that unit of measure is already set up.</span></span>
5. <span data-ttu-id="cc3a7-117">Seleccione la acción **Nuevo**.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-117">Choose the **New** action.</span></span> <span data-ttu-id="cc3a7-118">Se insertará una nueva línea vacía.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-118">A new empty line is inserted.</span></span>
6. <span data-ttu-id="cc3a7-119">En el campo de **Código**, escriba el nombre de la unidad de medida.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-119">In the **Code** field, enter the name of the unit of measure.</span></span> <span data-ttu-id="cc3a7-120">También, elija el campo para seleccionar de los códigos de la unidad de medida que figuren en la base de datos.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-120">Alternatively, choose the field to select from the unit of measure codes that are in the database.</span></span>
7. <span data-ttu-id="cc3a7-121">En el campo **Cdad. por unidad medida**, especifique cuántas unidades de la unidad de medida base contiene la nueva unidad de medida.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-121">In the **Qty. per Unit of Measure** field, enter how many units of the base unit of measure the new unit of measure contains.</span></span>
8. <span data-ttu-id="cc3a7-122">Repita los pasos 5 a 7 para configurar todas las unidades de medida alternas que desea utilizar en distintos procesos para este producto.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-122">Repeat steps 5 through 7 to set up all the alternate units of measure that you want to use in different processes for this item.</span></span>

<span data-ttu-id="cc3a7-123">Ahora puede utilizar las unidades de medida alternas de los documentos de compra, producción y venta.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-123">You can now use the alternate units of measure on purchase, production, and sales documents.</span></span> <span data-ttu-id="cc3a7-124">Para obtener más información, consulte Introducir los códigos de unidad de medida predeterminada para las transacciones de compra y las transacciones de venta o Usar la unidad de medida de lote de fabricación.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-124">For more information, see Enter Default Units of Measure Codes for Purchase Transactions and Sales Transactions or Use the Manufacturing Batch Unit of Measure.</span></span>

## <a name="to-set-up-unit-of-measure-translations"></a><span data-ttu-id="cc3a7-125">Para configurar traducciones de unidades de medida</span><span class="sxs-lookup"><span data-stu-id="cc3a7-125">To set up unit of measure translations</span></span>
<span data-ttu-id="cc3a7-126">Cuando vende productos a clientes extranjeros, quizás le interese especificar la unidad de medida en el idioma del cliente.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-126">When you sell items to foreign customers, you may want to specify the unit of measure in the customer’s language.</span></span> <span data-ttu-id="cc3a7-127">Para ello, primero debe configurar las traducciones correspondientes de las unidades de medida.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-127">You can do this after you have set up the necessary unit of measure translations.</span></span>

1. <span data-ttu-id="cc3a7-128">Elija el icono ![bombilla que abre la función Dígame](media/ui-search/search_small.png "Dígame que desea hacer"), escriba **Uds. de medida** y luego elija el enlace relacionado.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-128">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Units of Measure**, and then choose the related link.</span></span>
2. <span data-ttu-id="cc3a7-129">Seleccione el código para el que desea configurar las traducciones y, a continuación, elija la acción **Traducciones**.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-129">Select the code for which you want to set up translations, and then choose the **Translations** action.</span></span>
3. <span data-ttu-id="cc3a7-130">En el campo **Cód. idioma**, seleccione la flecha desplegable para ver una lista de los códigos de idioma disponibles.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-130">In the **Language Code** field, select the drop-down arrow to see a list of available language codes.</span></span> <span data-ttu-id="cc3a7-131">Seleccione el código de idioma para el que desea especificar una traducción y, a continuación, elija el botón Aceptar para copiarlo en el campo.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-131">Select the language code for which you want to enter a translation, and then choose the OK button to copy the code to the field.</span></span>
4. <span data-ttu-id="cc3a7-132">En el campo **Descripción**, escriba el texto correspondiente.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-132">In the **Description** field, enter the appropriate text.</span></span>
5. <span data-ttu-id="cc3a7-133">Repita los pasos del 2 al 4 con los códigos de unidad de medida y los idiomas para los que desea especificar traducciones.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-133">Repeat steps 2 through 4 for the unit of measure codes and the languages for which you want to enter translations.</span></span>

1. <span data-ttu-id="cc3a7-137">Elija el icono ![bombilla que abre la función Dígame](media/ui-search/search_small.png "Dígame que desea hacer"), escriba **Productos** y luego elija el enlace relacionado.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-137">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Items**, and then choose the related link.</span></span>
2. <span data-ttu-id="cc3a7-138">Abra la ficha de producto relevante para la que desea especificar un código de unidad de medida de compra o venta predeterminado.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-138">Open the relevant item card for which you want to specify a default sales or purchase unit of measure code.</span></span>
3. <span data-ttu-id="cc3a7-139">Para ventas, en la ficha desplegable **Facturación**, en el campo **Unidad medida venta**, abra la ventana **Unidades medida producto**.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-139">For sales, on the **Invoicing** FastTab, in the **Sales Unit of Measure** field, open the **Item Units of Measure** window.</span></span>
4. <span data-ttu-id="cc3a7-140">En compras, en la ficha desplegable **Reposición**, en el campo **Unidad medida compra**, abra la ventana **Unidades medida producto**.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-140">For purchasing, on the **Replenishment** FastTab, in the **Purch. Unit of Measure** field, open the **Item Units of Measure** window.</span></span>
5. <span data-ttu-id="cc3a7-141">Seleccione el código que desea configurar como la unidad de medida predeterminada para ventas o compras, respectivamente y elija el botón **Aceptar**.</span><span class="sxs-lookup"><span data-stu-id="cc3a7-141">Select the code you want to set up as the default unit of measure for sales or purchasing respectively, and then choose the **OK** button.</span></span>
